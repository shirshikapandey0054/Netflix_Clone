# Netflix_Clone

Netflix Clone (HTML & CSS)

A responsive Netflix landing page clone built using pure HTML5 and CSS3, focused on recreating the original Netflix UI with clean layout, modern styling, and responsive design principles.

Project Overview

This project is a frontend-only clone of the Netflix homepage. It was created to practice web layout design, CSS styling, and responsive UI development without using any JavaScript or frameworks.

The goal of this project is to closely match the look and feel of Netflix while keeping the code simple, readable, and beginner-friendly.
